# v0.3.0 Testing Checklist

## 测试准备
1. [ ] 在 Obsidian 中按 Ctrl/Cmd + R 重新加载插件
2. [ ] 确认插件版本显示为 0.3.0
3. [ ] 打开一个测试笔记

## Phase 1: 播放速度控制测试

### 速度选择器 UI
- [ ] 控制条显示速度选择器（🎬图标 + 倍速下拉）
- [ ] 速度选择器位置在音色选择器和对话模式之间
- [ ] 下拉菜单包含 6 个选项：0.5x, 0.75x, 1.0x, 1.25x, 1.5x, 2.0x
- [ ] 默认选中 1.0x

### 普通 TTS 模式速度控制
- [ ] 选择 0.5x，点击播放，语音速度明显变慢
- [ ] 选择 1.5x，点击播放，语音速度明显加快
- [ ] 选择 2.0x，点击播放，语音速度最快
- [ ] 播放中切换速度，立即生效
- [ ] 音质无明显失真

### 对话模式速度控制
- [ ] 点击对话按钮，生成对话
- [ ] 选择 0.75x，播放速度变慢
- [ ] 选择 1.25x，播放速度加快
- [ ] 播放中切换速度，立即生效
- [ ] 多个音色的速度都改变

### 速度设置持久化
- [ ] 选择 1.5x，重新加载插件
- [ ] 速度选择器显示 1.5x（设置已保存）

## Phase 3: 对话模板测试

### 配置 Modal UI
- [ ] 点击 💬 按钮，配置 Modal 弹出
- [ ] Modal 显示 5 个模板卡片：
  - [ ] 📖 单人讲解
  - [ ] 💬 双人对话
  - [ ] 📚 三人课堂
  - [ ] ⚔️ 双人辩论
  - [ ] 🎙️ 访谈模式
- [ ] 每个卡片显示图标、名称、描述
- [ ] 点击卡片，卡片高亮显示（selected 状态）
- [ ] 只能选择一个模板

### 风格选择 UI
- [ ] Modal 显示 3 个风格选项：
  - [ ] 正式（语言规范、逻辑严密）
  - [ ] 轻松（语言通俗、比喻生动）
  - [ ] 幽默（加入幽默元素、俏皮话）
- [ ] 点击风格卡片，卡片高亮
- [ ] 只能选择一个风格

### Modal 交互
- [ ] 点击"取消"按钮，Modal 关闭，不生成对话
- [ ] 点击"开始生成"按钮，Modal 关闭，开始生成
- [ ] 选择不同模板和风格，设置被保存
- [ ] 再次打开 Modal，显示上次选择

### 模板 1: 单人讲解
- [ ] 选择"单人讲解"模板 + "正式"风格
- [ ] 生成的对话只有一个角色：[讲解者]
- [ ] 内容是直接讲解，无提问互动
- [ ] 语言规范、逻辑严密

### 模板 2: 双人对话
- [ ] 选择"双人对话"模板 + "轻松"风格
- [ ] 生成的对话有两个角色：[对话者A] 和 [对话者B]
- [ ] 对话平等、互动自然
- [ ] 语言通俗易懂

### 模板 3: 三人课堂
- [ ] 选择"三人课堂"模板 + "轻松"风格
- [ ] 生成的对话有三个角色：[讲师]、[好奇学生]、[批判学生]
- [ ] 讲师讲解，学生提问
- [ ] 好奇学生问基础问题，批判学生问深度问题

### 模板 4: 双人辩论
- [ ] 选择"双人辩论"模板 + "正式"风格
- [ ] 生成的对话有两个角色：[正方] 和 [反方]
- [ ] 双方观点对立、有来有回
- [ ] 语言有力、逻辑严密

### 模板 5: 访谈模式
- [ ] 选择"访谈模式"模板 + "轻松"风格
- [ ] 生成的对话有两个角色：[主持人] 和 [嘉宾]
- [ ] 主持人提问，嘉宾回答
- [ ] 访谈风格明显

### 风格差异测试
使用同一模板（如三人课堂）测试不同风格：

- [ ] **正式风格**：使用专业术语，语言规范
- [ ] **轻松风格**：通俗易懂，有比喻
- [ ] **幽默风格**：有幽默元素、俏皮话

三种风格应该有明显区别。
