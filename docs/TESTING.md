# 对话模式功能测试清单

## 测试环境准备

### 1. 插件安装
- [ ] 将 `main.js`, `styles.css`, `manifest.json` 复制到 Obsidian vault 的 `.obsidian/plugins/obsidian-tts/` 目录
- [ ] 重启 Obsidian 或重新加载插件
- [ ] 在设置中启用 "Obsidian TTS" 插件

### 2. API 配置
- [ ] 打开 Obsidian 设置 → TTS 设置
- [ ] 选择 TTS 引擎：**通义千问 TTS**
- [ ] 输入有效的 API Key
- [ ] 选择模型：`qwen3-tts-instruct-flash` 或 `qwen-long`
- [ ] 保存设置
