# Obsidian TTS Plugin v0.1.0 - Release Summary

## 🎉 发布概述

**版本**：v0.1.0
**发布日期**：2026-03-04
**主要特性**：💬 对话模式（Dialogue Mode）

这是 Obsidian TTS Plugin 的第一个正式版本，引入了革命性的**对话模式**功能，将 Markdown 文档转换为 NotebookLM 风格的三人对话，提供沉浸式学习体验。

## ✨ 核心功能亮点

### 💬 对话模式（全新）

1. **AI 智能生成**
   - 使用通义千问 API 实时生成对话
   - 自动分析文档内容和结构
   - 根据文档长度自适应对话详细程度

2. **三人角色系统**
   - **主讲人**（Ethan - 男性稳重）：讲解核心内容
   - **好奇学生**（Cherry - 女性活泼）：提出基础问题
   - **批判学生**（Serena - 女性理性）：提出深度问题

3. **智能长度适配**
   - 短文档（< 1000 字）：5 分钟对话
   - 中等文档（1000-3000 字）：15 分钟对话
   - 长文档（> 3000 字）：30 分钟对话

4. **完整的用户体验**
   - 实时进度显示（分析→生成→保存→准备）
   - 对话文件自动保存和管理
   - 支持重用现有对话或重新生成
   - 可手动编辑对话脚本

5. **多音色播放**
   - 自动根据角色切换音色
   - 流畅的音色过渡
   - 支持中英文混合内容

### 🎯 基础 TTS 功能

- 多引擎支持（Web Speech API、通义千问 TTS、OpenAI TTS）
- 智能 Markdown 解析
- 实时进度显示
- 11+ 种预设音色
- 自动语言检测
- 紧凑的单行 UI 布局

## 🔧 安装和使用

### 快速安装

1. 下载插件文件（main.js, styles.css, manifest.json）
2. 复制到 vault 的 `.obsidian/plugins/obsidian-tts/` 目录
3. 重启 Obsidian 并启用插件
4. 配置通义千问 API Key

### 使用对话模式

1. 打开要学习的 Markdown 笔记
2. 点击控制条上的 💬 按钮
3. 等待对话生成（显示进度）
4. 自动开始播放三人对话
5. 使用 ⏸ 暂停，⏹ 停止

### 对话文件管理

- **首次生成**：自动保存为 `文档名-对话.md`
- **再次使用**：选择"使用现有"或"重新生成"
- **手动编辑**：可直接编辑对话文件内容
